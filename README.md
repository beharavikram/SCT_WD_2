# Stopwatch Web Application

A simple stopwatch web application built using HTML, CSS, and JavaScript.

## Features

- **Start:** Begin the stopwatch timer.
- **Stop:** Pause the stopwatch timer.
- **Reset:** Reset the stopwatch timer to zero.
